# rmi

rmi，remote method invocation/远程调用方法，它简化了多台计算机上的java应用之间的通信（相较于socket），是分布式对象软件包。所有可以被远程调用的对象都必须实现java.rmi.Remote接口，所有可以被远程调用的对象都必须继承java.rmi.UnicastRemoteObject类
